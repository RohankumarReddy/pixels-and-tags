## **Overview**

The **Climate Crisis News API** provides real-time access to global climate and environment news articles by scraping multiple verified news outlets.  
It aggregates the latest updates from trusted sources such as **The Guardian**, **BBC**, **New York Times**, and **United Nations**, offering a simple and structured JSON response.

---

## **Features**

- Fetches news articles containing “climate” from multiple sources  
- Returns aggregated JSON data of latest articles  
- Filter by specific newspaper  
- Generate random sets of climate-related articles  
- Built with lightweight and efficient **Node.js + Express + Cheerio**

---

## **Tech Stack**

| Component | Technology | Description |
|------------|-------------|--------------|
| **Runtime** | [Node.js](https://nodejs.org/) | JavaScript runtime for backend APIs |
| **Framework** | [Express.js](https://expressjs.com/) | Fast, unopinionated backend framework |
| **HTTP Client** | [Axios](https://axios-http.com/) | Handles external requests to fetch HTML |
| **Web Scraper** | [Cheerio](https://cheerio.js.org/) | Parses and extracts structured data from webpages |
| **Output Format** | JSON / JSONP | API responses formatted for cross-origin compatibility |

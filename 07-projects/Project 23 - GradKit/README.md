# GradKit
**Your AI Growth Companion**

GradKit is a modern platform that helps learners and professionals stay industry-ready through AI-driven insights, mock interviews, and automated career tools.

## ⚡ Features

- **Industry Insights:** Real-time data on trending technologies and job market shifts.
- **AI Mock Interviews:** Practice technical and behavioral questions with instant scoring.
- **Smart Resume Builder:** Generate ATS-friendly resumes and cover letters using AI.
- **Personalized Learning Feed:** Weekly curated updates tailored to your industry.
- **Weekly Digest:** Automated summaries of market and tech updates.

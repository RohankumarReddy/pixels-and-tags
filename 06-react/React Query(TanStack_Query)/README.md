### TanStack Query 

This project uses **TanStack Query** to handle API calls in React.

* **GET data** using `useQuery`
* **POST data** using `useMutation`
* **Refresh data** using `invalidateQueries`

TanStack Query automatically manages loading, errors, and caching, making API handling easier and cleaner.

**Summary:**

* `useQuery` → fetch data
* `useMutation` → send data
* `invalidateQueries` → refetch updated data

That’s it.
